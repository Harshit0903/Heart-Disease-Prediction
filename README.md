# Heart Disease Prediction Model  

This project predicts heart disease using machine learning algorithms.  

## Technologies Used  
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  

## Models Implemented  
- Logistic Regression  
- Naive Bayes  
- K-Nearest Neighbors (KNN)  
- Decision Tree  
- Random Forest (Best Accuracy: 86.89%)
